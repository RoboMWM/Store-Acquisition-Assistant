This app helps assist a user in performing acquisitions of products from the Microsoft Store on PC, e.g. acquiring Windows Phone apps.

This used to be possible in the native store client app.

Appx download is in the [Releases](https://github.com/RoboMWM/Store-Acquisition-Assistant/releases) section.

If it ever happens to be over a year since the last release, let me know so I can reupload an installable one.
